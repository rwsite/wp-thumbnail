# Wordpress thumbnail plugin. Generate and cache thumbnail on fly.


1. With WEBP support
2. On fly
3. Fast


How to use it (see functions.php):

###### Get SRC
> kama_thumb_src($args = [], $src = 'notset');

###### Get Img

> kama_thumb_img( $args = [], $src = 'notset' )

###### Get Img link
> kama_thumb_a_img( $args = [], $src = 'notset' )
